# Document Intelligence API

A FastAPI backend service for intelligent document processing.

## Features

- Upload documents through an API endpoint
- Extract text from uploaded files
- Split documents into manageable chunks
- Store and retrieve relevant document chunks
- Search documents using vector-based retrieval
- Generate answers using Retrieval-Augmented Generation (RAG)

## Technologies

- Python
- FastAPI
- Pydantic
- Uvicorn
- Vector Database / Embeddings
- RAG Architecture
